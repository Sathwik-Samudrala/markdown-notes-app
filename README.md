# 📝 Markdown Notes App

A full-stack Notes Application built using React and Node.js with live Markdown preview.

---

## 🚀 Features

- Create, edit, and delete notes  
- Live Markdown preview  
- Search notes instantly  
- Dark mode toggle  
- Auto-save functionality  

---

## 🛠 Tech Stack

Frontend:
- React
- Axios
- React Markdown

Backend:
- Node.js
- Express

Other:
- REST API
- CSS

## ⚙️ Installation

Clone the repository:

git clone https://github.com/Sathwik-Samudrala/markdown-notes-app.git  
cd markdown-notes-app  

---

Run Backend:

cd backend  
npm install  
node server.js  

---

Run Frontend:

cd frontend  
npm install  
npm start  

---

## 🌐 Running

Frontend: http://localhost:3000  
Backend: http://localhost:5000  

---

## 💡 About Project

This is a full-stack application that allows users to create and manage notes with live markdown preview. It demonstrates React frontend integration with a Node.js backend using REST APIs.
